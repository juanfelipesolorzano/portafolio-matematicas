# portafolio-matematicas

Este repositorio contiene parte del trabajo académico realizado durante el semestre 2026-1 en las áreas de análisis, probabilidad y matemática estructural.

El material incluye talleres, apuntes, lecturas y ejercicios desarrollados como parte de un proceso de preparación académica y reincorporación gradual al entorno universitario durante el 2026-1.

Parte de este trabajo fue realizado asistiendo a cursos con autorización de las profesoras correspondientes.
